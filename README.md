# text-generator

# 🧠 AI Text Generator (Streamlit App)

A clean and interactive **Text Generator Web App** built using **Python** and **Streamlit**.
This project demonstrates how to create a simple AI-style text generation system with a user-friendly interface.

---

## 🚀 Demo Preview

✨ Generate text from your custom prompt
🎛️ Adjust output length
⚡ Instant results in a clean UI

---

## 📌 Features

* 📝 Prompt-based text generation
* 🎚️ Adjustable text length slider
* ⚡ Fast and lightweight (no heavy ML required)
* 🎨 Simple and clean UI using Streamlit
* 📂 Well-structured and beginner-friendly code

---

## 🏗️ Project Structure

```
text-generator-app/
│
├── app.py                  # Streamlit frontend
├── generator.py            # Text generation logic
├── requirements.txt        # Dependencies
├── README.md               # Project documentation
└── utils/
    └── helpers.py          # Helper functions
```

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/text-generator-app.git
cd text-generator-app
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the App

```bash
streamlit run app.py
```

---

## 🖥️ How It Works

1. User enters a **prompt**
2. Selects desired **text length**
3. Clicks **Generate**
4. App produces generated text using Python logic

---

## 🧩 Core Components

### 🔹 `app.py`

* Handles UI using Streamlit
* Takes user input and displays output

### 🔹 `generator.py`

* Contains logic for generating text
* Uses simple random-based approach

### 🔹 `helpers.py`

* Utility functions like text cleaning
